class Bstation[source]

Bstation commands group definition. 171 total commands, 13 Sub-groups, 1 group commands Repeated Capability: Stream, default value after init: Stream.Nr1

preset() → None[source]
# SCPI: [SOURce<HW>]:BB:W3GPp:BSTation:PRESet
driver.source.bb.w3Gpp.bstation.preset()

The command produces a standardized default for all the base stations. The settings correspond to the *RST values specified for the commands. All base station settings are preset.

preset_with_opc() → None[source]
# SCPI: [SOURce<HW>]:BB:W3GPp:BSTation:PRESet
driver.source.bb.w3Gpp.bstation.preset_with_opc()

The command produces a standardized default for all the base stations. The settings correspond to the *RST values specified for the commands. All base station settings are preset.

Same as preset, but waits for the operation to complete before continuing further. Use the RsSmw.utilities.opc_timeout_set() to set the timeout value.
