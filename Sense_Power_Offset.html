class Offset[source]

Offset commands group definition. 2 total commands, 1 Sub-groups, 1 group commands

get(channel=<Channel.Default: -1>) → float[source]
# SCPI: SENSe<CH>:[POWer]:OFFSet
value: float = driver.sense.power.offset.get(channel = repcap.Channel.Default)

Sets a level offset which is added to the measured level value after activation with command OFFSet:STATe. The level offset allows, e.g. to consider an attenuator in the signal path.

param channel

optional repeated capability selector. Default value: Nr1 (settable in the interface ‘Sense’)

return

offset: float Range: -100.0 to 100.0, Unit: dB
